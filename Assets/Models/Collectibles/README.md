## File Formats

Each model is provided in the following formats:
- `.fbx` - For Unity import (primary format used in this project)
- `.blend` - Original Blender files (if available)
- `.obj` + `.mtl` - Universal 3D format

## Notes

- **Materials:** The models are provided without materials. You will need to create and apply materials in Unity.
- **Scale:** Low-poly style, optimized for game performance.
- **Usage:** These assets are CC0 licensed, meaning you can use them freely without attribution (though attribution is always appreciated).

## Integration in Unity

1. Import the FBX files into Unity (already done - files are in this folder)
2. Create materials for the models in Unity
3. Set up prefabs for each collectible type
4. Add the `Collectible` script component to each prefab
5. Configure collision detection (add Collider with `isTrigger = true`)
